# vue-mask
使用vue2实现的H5端口罩头像生成器，可以将微信头像、QQ头像等等替换成戴口罩、护目镜的合成头像

手机端体验地址：
https://brenner8023.gitee.io/vue-mask/

食用代码：
```shell
# project setup
npm i

# Compiles and hot-reloads for development
npm run serve

# Compiles and minifies for production
npm run build
```
