# vue-mask
使用vue2写的H5口罩头像生成器

h5访问：
https://brenner8023.gitee.io/vue-mask/

食用
```shell
# project setup
npm i

# Compiles and hot-reloads for development
npm run serve

# Compiles and minifies for production
npm run build
```
